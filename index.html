O que é HTML?

HTML é uma das linguagens de marcação mais conhecidas e utiliza um formato simples de código, que pode ser gerado sem a ajuda de aplicativos especiais.

Para criar um documento HTML, tudo o que você precisa é de um editor de texto ASCII como o NotePad (Bloco de Notas), por exemplo. Se você precisar verificar a formatação do documento durante a criação, pode usar qualquer navegador, pois todos eles são capazes de interpretar o código.

Frames (depreciada)


Para criação de frames usa se a tag frameset, ele é utilizado como a tag body em questão de preenchimento, pode ser definido seu tamanho a partir dos atributos cols e rows que representam a largura e a autura do frame. Exemplo: cframe cols="25%", *,25%"> (o * representa o tamanho que falta para completar a tela, no caso 50%)

Os atributos da tag frame são src="url", marginwidth="margin largura", marginheight="margin altura", scrolling="add uma barra de rolagen ao frame" e noresize="inpede que o user redirecione o frame"


Palavra Chave (Meta)


As palavras chaves não sao exibitas para o usuario final, elas descrevem as definições da pagina e ajudam na divulgação, ela é representada pela tag meta e deve ser inserida na head da pagina. Exemplo:

Seu valor depende do valor dos atributos name="" ou http-equiv=""


    Os atributos do valor nome são:

  1. author: autor do documento
  2. descriotion: descreve o documento
  3. keywords: palavra chave que descreve o documento
  4. generator: define o programa utilizado para gerar o documento
  5. robots: tags para facilitar a busca pelos sistemas de busca
  6. copyright: direitos autorais

  1. content type: define o tipo do documento (html/text)
  2. expires: define o tempo que o documento será obsoleto
  3. refresh: define o tempo de recarga da pagina
  4. set-cookie: define o valor de um cookie
